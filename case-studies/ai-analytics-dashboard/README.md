# AI Analytics Dashboard

## Working Idea

A dashboard demo that pulls performance metrics, visualizes trends, and uses an LLM to generate clear monthly insights, recommendations, and review notes.

## Demo Surface

- KPI tiles.
- Trend charts.
- AI-generated monthly summary.
- Recommendation queue.
- Client-ready report view.

## Stack To Demonstrate

- React-style dashboard interface.
- API ingestion pattern.
- SQL analytics layer.
- Claude/OpenAI narrative generation.
- Multi-client reporting template.

# Data Platform Control Tower

## Working Idea

A pipeline observability demo that shows ingestion status, data quality, lineage, freshness, backfills, and business-facing metrics.

## Demo Surface

- Pipeline health dashboard.
- Bronze/silver/gold flow.
- Data quality checks.
- Backfill and retry status.
- Stakeholder-ready summary.

## Stack To Demonstrate

- SQL and Python ETL/ELT.
- Airflow or Dagster orchestration concepts.
- dbt-style transformation layers.
- Warehouse/lakehouse modeling.
- Monitoring and alerting.

## Open Decisions

- Pick a domain: payments, marketing analytics, ecommerce operations, or product events.
- Decide whether to make it a pure visual dashboard or a simulated interactive control plane.

# Document Intelligence Workbench

## Working Idea

A document processing demo that shows extraction, schema validation, confidence gates, field-level evidence, and human review before output is accepted.

## Demo Surface

- Document intake queue.
- Extracted fields.
- Confidence markers.
- Validation errors.
- Review and approve workflow.

## Stack To Demonstrate

- Python document pipeline.
- OCR/document extraction pattern.
- LLM structured extraction.
- JSON schema validation.
- AWS-style storage and processing architecture.

## Open Decisions

- Pick a general document class: invoices, claims, intake forms, contracts, or operations documents.
- Decide whether to use static sample documents or generate synthetic docs visually.
